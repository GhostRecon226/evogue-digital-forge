## Update Google Search Console Verification Token

Replace the placeholder `google-site-verification` meta tag in `index.html` with the real token Google provided.

### Change

**File:** `index.html` (line 8)

Replace:
```html
<meta name="google-site-verification" content="googleb0628a439996e231" />
```

With:
```html
<meta name="google-site-verification" content="Ycz5PXm4ARVRP7WUUywjZWli13bpslDvuheHpATpiP8" />
```

### After approval
1. Click **Publish → Update** in Lovable so the change goes live on `evogue.com.ng`.
2. Return to Google Search Console and click **Verify**.
